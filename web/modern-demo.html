3. How It Works

File Organization:

/assets/
  styles/
    tokens.css          ‚Üê Design tokens only
  components/
    index.js            ‚Üê Main export
    base.js             ‚Üê Base class & utilities
    Card.js             ‚Üê Card component
    Button.js           ‚Üê Button component
    DashboardLayout.js  ‚Üê Layout component

Usage (Simple HTML):

<!-- Import components -->
<script type="module">
  import '/assets/components/index.js';
</script>

<!-- Use them in HTML -->
<wr-card title="My Card" variant="glass">
  Content here
</wr-card>

<wr-button variant="primary" icon="rocket">
  Click Me
</wr-button>

4. Benefits

ES Modules work natively in browsers.

No Webpack, Vite, or bundler required.

Shadow DOM scopes styles.

CSS can't conflict between components.

Browser only loads what you import.

Unused components aren't downloaded.

Web Components are a standard.

Works with any framework or none.

Each component is self-contained.

One file per component.

Native browser API.

Will work forever.
